Apollo Saturn V

This is a great set for it's price. It is a challenging build that may be difficult for inexperienced builders but once complete makes a great display set

10/10 - $110

Titanic

An iconic build that while extraordinarily overpriced is an amazing set for every lego enthusiast. The build takes a while and has some repitition but there are enough details so that it never gets boring. It took me all of April break to build it so I would say it is well worth it if you are really into LEGO.

8/10 - $680

Assault on Hoth

One of my favorite sets in terms of playablity. The modular aspects of this set make it a great foundation for epic battle scenes and just general playing.

7/10 - $250
